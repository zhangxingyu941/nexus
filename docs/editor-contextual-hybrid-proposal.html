交互稿里需要补充同步失败时的反馈方式。

林然 · 刚刚

进度条用蓝色，避免和保存成功的绿色混淆。

01 / 默认状态写作优先

右侧上下文面板默认关闭，正文保持 760px 阅读宽度；区块操作仅在悬停或聚焦时显露。

02 / 触发方式围绕当前区块

选中任务块后打开右侧面板；切换到普通段落或按 Esc 时关闭，避免工具永久占据画布。

03 / 颜色语义颜色只表达状态

绿色用于保存成功，蓝色用于任务进度，琥珀色用于临近截止，红色只用于同步失败与破坏性操作。

04 / 异常反馈失败可见且可恢复

同步失败同时出现在顶栏状态和页面通知中，明确“本地修改已保留”，并提供重试入口。
