## M7.1 文档权限

状态：功能实现完成，M7.1 目标 E2E 通过。

已完成：

- 文档直接路由 `/documents/:documentId`，未认证进入登录流程，无权访问统一返回 404。
- 私有/团队/链接文档策略、显式 `editor`/`viewer` 权限和 owner 管理权限。
- `created_by` 历史回填固定使用工作区最早加入的 owner（按 `created_at`、`user_id` 排序）；该字段只影响作者元数据，不削弱任何 owner 的管理权限。
- 文档读写、历史读取/恢复、文件附件和协作 WebSocket 统一使用文档授权判定。
- 工作区目录过滤不可访问私有文档；远程模式只保存活动文档。
- 远程文档新建、复制、删除 API；删除后服务端为当前用户选择仍可访问的活动文档。

## M7.2 匿名链接分享

状态：功能实现完成，M7.2 目标 E2E 通过。

已完成：

- 仅工作区 owner 可创建、查看状态、重新生成和关闭分享链接；每篇文档最多一个未撤销链接。
- 默认有效期 24 小时，可选 1 小时、7 天、30 天或自定义时间，自定义上限 365 天。
- 令牌使用 256 bit 随机值，数据库只保存域隔离 HMAC；完整 URL 仅在创建或重新生成响应中返回一次。
- 未登录访问 `/share/:token` 时只渲染标题、公开块和已授权附件，不加载工作区壳、评论、历史、成员、任务元数据或 Yjs。
- 未知令牌返回 404；已过期、已撤销和被重新生成替换的令牌返回 410。
- 附件通过 5 分钟签名代理读取，每次请求重新验证分享状态；关闭或重新生成后旧附件 URL 立即返回 410。
- 从 `link` 策略切换到其他模式时在同一事务撤销活动链接；创建、重新生成、关闭、访问及拒绝均写脱敏审计。
- 匿名响应不包含内部文档 ID、附件 object key、评论或任务私有字段。
